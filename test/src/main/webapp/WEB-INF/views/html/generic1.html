명품 주얼리

온라인 주얼리 쇼핑몰 종로3가닷컴은 2021년 상담한 고객들이 선호하는 해외 명품 브랜드를 분석한 결과 이같이 나타났다고 30일 밝혔다.

종로3가닷컴에서 상담한 브랜드 중 베스트5를 꼽아보면 1위부터 티파니, 해리 윈스턴, 까르띠에, 반클리프 아펠, 불가리로 순으로 집계됐다.
