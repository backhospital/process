가치 측정이란?

스마트기기의 카메라 디바이스를 활용하여 정밀하게 촬영하면 해당 명품의 가치를 판별할 수 있도록 하는 서비스를 제공합니다.

-- 사용방법 --

1단계 하단 버튼을 클릭하여 카메라를 활성화 한다.

2단계 카메라와 명품간의 거리를 최소화하며, 약 10Cm 내외로 가장 선명한 영상을 확보할 수 있도록 조절한다.

3단계 스마트기기의 흔들림을 최소화하며 호흡을 잠시 멈추고 촬영 버튼을 클릭한다.

4단계 판별에 적합한 영상물이 확보되면 자동으로 명품의 가치에 대한 측도를 산출하여 표시한다.

명품 가치측정 시작

나의 명품의 가치 측도를 확인하기 위해서 클릭해주세요.
